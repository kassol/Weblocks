# Weblocks

Playful browser construction world. V1 core is renderer-independent; Free Build uses Vite + Three.js.

## Commands

```sh
pnpm install
pnpm lint
pnpm typecheck
pnpm test
pnpm build
pnpm preview
pnpm dev
```
